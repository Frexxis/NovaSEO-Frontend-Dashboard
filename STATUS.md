## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/Frexxis/NovaSEO-Frontend-Dashboard.git
cd NovaSEO-Frontend-Dashboard

# Install dependencies
pnpm install

# Start development server
pnpm dev
```
